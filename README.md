# portal-leqiu-lab

## 项目简介

portal-leqiu-lab 是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，而是作为一个通用容器，用于存放各类静态 HTML 文件及其相关资源。

这些页面可能包括工具页面、信息展示、实验性界面或学习笔记等，内容多样，用途不限。

## 目录说明

```
portal-leqiu-lab/
├── pages/          # 存放各独立 HTML 页面及资源
├── assets/         # 公共资源（如样式、脚本、图片等）
└── README.md       # 本文件
```

- `pages/`：每个子目录或文件对应一个独立页面，页面间互不依赖。
- `assets/`：存放可在多个页面中复用的公共资源，方便统一管理。

## 页面归档说明

本仓库中的页面均为独立归档，每个页面包含完整的 HTML 结构及相关资源（CSS、JavaScript、图片等）。页面之间没有逻辑依赖，可以单独打开使用。

归档内容可能随时更新或调整，不保证所有页面长期可用或保持不变。
